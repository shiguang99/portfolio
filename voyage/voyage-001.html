AI工具赋能副业：从0到1的实战复盘

Claude Code OpenClaw

项目背景

2024年加入生财有术社区，参与官方实战航海项目。 目标是用 AI 工具在3个月内实现副业月入3000元，并将整个过程完整记录下来， 供社区成员参考和复用。

作为一个没有编程背景的建筑工程师，我想验证一个核心命题： 普通人借助 AI 工具，能否独立开发出可商业化的产品？

实战过程

从选题调研、工具选型、产品开发到推广运营，全程使用 AI 工具辅助决策和执行。 重点记录了 Claude Code 在代码生成、OpenClaw 在内容创作方面的实际效果和使用技巧。

整个过程分为三个阶段：

阶段一：选题与验证 — 用 AI 工具做市场调研，筛选有需求的细分赛道
阶段二：产品开发 — Claude Code 辅助编码，从零完成小程序开发
阶段三：推广运营 — OpenClaw 辅助内容创作，探索流量获取渠道
